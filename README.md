# 棚卸し表 — Golot

店舗向け棚卸し管理Webアプリ（シングルHTMLファイル）

## URL
GitHub Pagesで公開: https://yoshi-5772.github.io/my-first-project/

## 機能
- **月別保存**: ◀ ▶ で月を切り替え、月ごとにデータを保存。新月は前月の品名・単価・数量を引き継ぎ
- **部門タブ**: 精肉 / 食材 / 調味料 / 酒/飲材 / 備品/包材
- **サマリータブ**: 部門別合計・総合計を一覧表示
- **税率自動計算**: 食料品（精肉・食材・調味料）8%、酒類・備品 10%。税抜入力→税込自動変換
- **リアルタイム集計**: 品名・単価・数量を変更すると小計・合計が即座に反映
- **品目追加・削除**: フォームから手動追加、削除ボタンで削除
- **写真OCR**: Tesseract.jsで写真から文字を読み取り品目追加補助
- **CSVダウンロード**: 部門別小計・合計付きCSV（ファイル名: 棚卸し表_YYYY-MM.csv）
- **印刷**: 全タブに印刷ボタン（印刷時は入力欄・矢印を非表示）

## データ保存
ブラウザのlocalStorageに月別キー `inv_YYYY-MM` で保存。
